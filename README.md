# Phillip's Pressure Warriors

This repository contains a small static website for Phillip's Pressure Warriors, a pressure washing business serving Fairfield and Mt. Pleasant, IA.

## What’s included
- `index.html` — homepage with service overview, benefits, FAQ, and contact form
- `prices.html` — pricing, fee details, and contract policy
- `locations.html` — service area details
- `appointments.html` — appointment booking form
- `about.html` — company mission, values, and hiring information
- `testimonials.html` — customer reviews and trust statements
- `hours.html`, `tickets.html`, `staffportal.html`, `staffaccess.html`, `staffpanel.html`, `ownership.html`, `supervisor.html` — support and staff access pages
- `styles.css` — shared website styles for consistent design and mobile responsiveness

## Usage
Open any page in a browser, starting with `index.html`.

## Improvements included
- shared stylesheet for consistent look and easier maintenance
- modern mobile-friendly layout
- improved navigation and footer across pages
- clearer service presentation and contact information
- better readability and page metadata for search friendliness
